# mHealth-Schwimm-Atemuebungen
Wir helfen ambitionierten Schwimmern bessere Atemkontrolle und Ausdauer zu erlangen, indem wir ihnen schwimmspezifischen Atemübungen zur Verfügung stelle, ohne dass sie sportmedizinisches Vorwissen benötigen.

---

## Testumgebung starten

#### Vorraussetzungen
`node --version` -> `v22.22.2`<br>
und<br>
`yarn --version` -> `1.22.22` <br>
<br>


Dependencies installieren:
```bash
yarn install
```

Starten mit:
```bash
yarn start
```
Und dann den QR-Code scannen und mit der Expo Go App öffnen (funktioniert nicht im edoram W-Lan)

oder:<br>
`yarn start -w` für browser<br>
`yarn start -i` für iPhone Simulator (nur auf Mac)<br>
`yarn start -a` für Android Emulator (mit Android Studio)<br>
